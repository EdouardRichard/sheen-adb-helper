## 最终范围

完成手动连接与配对码回退、单活跃 Session、设备档案/概览、原始 Shell、只读进程、前台 Logcat、设置/隐私、SAF 导出、清除本地数据和进程内脱敏诊断。未交付应用管理、文件管理/传输、二维码配对、发现、后台采集或提权能力。

## 已知限制与延期

- 旧式 `:5555` 和 ROM 命令/字段为尽力兼容；无线调试端口可能变化。
- 配对库 API 会短暂构造不可原地覆盖的 JVM `String`。
- 应用管理在后续 v0.02 另行定义并完成；其完成不回写扩大 v0.01 范围。

## 证据缺口与阻断

- 证据缺口以 [`verification.md`](verification.md) 为准，不能从总体通过反推逐项有证据。
- ADR 0004 的许可证冲突在取得白名单授权或替换实现并完成回归前持续阻断对外发布合规。
- 本归档纠正 legacy 结项记录末句曾出现的“许可证可忽略”矛盾表述：许可证阻断不可忽略，功能结项与发布合规必须分开。
